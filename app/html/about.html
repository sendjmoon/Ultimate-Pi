ABOUT

Ultimate Pi is an application inspired by a group of developers who share a curiosity towards technology. Our goal is to create a platform that acts as a bridge between a user and their device. This can commonly be referred to as a remote control but often times it can (and will be) misplaced. In the event of such unfortunate circumstance, Ultimate Pi's true utility can be seen.

By running a Node server on the Raspberry Pi, Ultimate Pi is able to provide a web interface allowing users to send infrared commands via HTTP requests to a REST API. Heroku is used to handle the HTTP requests that are then forwarded to the Pi directly.

This project is only in it's beginning stages, but has the potential to reach extraordinary heights. In our features page more information regarding our next steps can be found.
